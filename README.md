# 📊 Retail Sales & Profitability Insights Dashboard

## 📖 Project Overview
This project analyzes retail sales data to identify key factors affecting profitability, customer behavior, and business performance.

---

## 🎯 Objectives
- Analyze sales trends over time
- Identify profitable and loss-making products
- Understand the impact of discounts on profit
- Evaluate regional and segment-wise performance

---

## 🛠 Tools & Technologies
- Python (Pandas, Matplotlib, Seaborn, plotly)
- SQL (PostgreSQL)
- Power BI

---

## 📊 Dashboard Preview
![Dashboard](dashboard.png)

---

## 📈 Key Insights
- Sales show seasonal trends with peak periods
- High discounts negatively impact profitability
- Some products generate high sales but result in losses
- Corporate segment contributes the highest profit
- Certain regions are underperforming

---

## 🔍 Root Cause Analysis
- Excessive discounting reduces profit margins
- High operational costs affect certain categories
- Inefficiencies in specific regions lead to losses

---

## 💡 Recommendations
- Optimize discount strategy
- Focus on high-margin products
- Improve performance in loss-making regions
- Target corporate and repeat customers

---

## 🏁 Conclusion
This project highlights the importance of balancing sales growth with profitability using data-driven decision making.
